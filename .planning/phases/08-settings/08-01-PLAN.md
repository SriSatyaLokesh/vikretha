**After rendering, do the following (wiring):**

**1 — Load staff list:**
Define `async function _loadStaff()`:
- `const configRef = doc(db, 'shops', SHOP_ID, 'config')`
- `const snap = await getDoc(configRef)`
- Extract `emails = snap.exists() ? (snap.data().authorized_emails || []) : []`
- Render into `#staff-list`:
  - If empty: `<p class="settings-empty">No staff added yet.</p>`
  - Otherwise: one `<div class="staff-email-item">` per email with:
    - `<span class="staff-email-text">{email}</span>`
    - `<button class="staff-remove-btn" data-email="{email}" ${email === currentEmail ? 'disabled title="Cannot remove your own account"' : ''}>Remove</button>`
- On error: show `<p class="settings-error">Could not load staff list.</p>`

Call `_loadStaff()` immediately after rendering.

**2 — Remove email:**
Add `click` listener on `#staff-list` (delegated):
- Match `e.target.closest('.staff-remove-btn')`
- If found and not disabled:
  - `const emailToRemove = btn.dataset.email`
  - `if (!confirm('Remove access for ' + emailToRemove + '?')) return`
  - Disable the button (set `btn.disabled = true; btn.textContent = '…'`)
  - `await updateDoc(doc(db, 'shops', SHOP_ID, 'config'), { authorized_emails: arrayRemove(emailToRemove) })`
  - On success: call `_loadStaff()` to refresh list
  - On error: re-enable button, alert `'Could not remove: ' + err.message`

**3 — Add email:**
Add `submit` listener on `#add-email-form`:
- `e.preventDefault()`
- `const email = container.querySelector('#add-email-input').value.trim().toLowerCase()`
- If `!_isValidEmail(email)`: show error `'Please enter a valid email address.'` in `#add-email-error` and return
- Hide error, disable submit button (`textContent = 'Adding…'`)
- `await updateDoc(doc(db, 'shops', SHOP_ID, 'config'), { authorized_emails: arrayUnion(email) })`
- On success: clear input, call `_loadStaff()`, re-enable button
- On error: show error in `#add-email-error`, re-enable button

**4 — Sign out:**
Add `click` listener on `#sign-out-btn`:
- Disable button, set text `'Signing out…'`
- `await signOut(auth)` (onAuthStateChanged in app.js redirects to login)
- On error: re-enable button, text back to `'Sign Out'`
