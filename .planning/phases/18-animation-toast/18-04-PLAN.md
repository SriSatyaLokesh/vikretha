<objective>
Refactor the entire CSS colour system to flow from three design primitives — foreground, background, and accent — so dark mode is self-maintaining and never requires component-level overrides.

Purpose: Eliminate the 25-rule `[data-dark="true"]` component override block (added in Phase 17) by replacing raw `--slate-N` usages in component CSS with semantic surface tokens that automatically invert in dark mode.
Output: A single source-of-truth for colour behaviour — change three variables and the whole app re-themes correctly in both light and dark mode.
</objective>

<what-built>
## Three Design Primitives (`:root`)

```css
--color-fg:     #0f172a;   /* near-black  — text & icons (light mode) */
--color-bg:     #ffffff;   /* white       — page canvas  (light mode) */
--color-accent: #f97316;   /* orange      — brand accent (both modes) */
```

Flip `--color-fg` / `--color-bg` in `[data-dark="true"]` to invert. Change `--color-accent` to rebrand.

## Surface Scale (3 tokens)

| Token | Light | Dark |
|---|---|---|
| `--surface-subtle` | `var(--slate-50)` | `rgba(255,255,255,0.04)` |
| `--surface-hover`  | `var(--slate-100)` | `rgba(255,255,255,0.07)` |
| `--surface-press`  | `var(--slate-200)` | `rgba(255,255,255,0.12)` |

## Component CSS

20 raw `var(--slate-50/100/200/600)` references in component CSS replaced with the appropriate surface token:
bar chart bars, monthly stat boxes, top-products items, btn-secondary hover, btn-ghost hover, badge-slate, role-member, seg-toggle tray, inv-table th/tr hover, inv-card active, export dropdown hover, bottom-sheet-close hover, sync-badge, staff-email-item, reports-table rows, rpt-close-x, rpt-edit-toggle-btn, rpt-picker-item hover.

## Dark-Mode Override Section

Trimmed from **25 rules → 6 rules**. The 6 retained rules cover only overrides that cannot be expressed via the three surface tokens:
- Segmented control active pill (explicit lift above tray)
- Alert borders (hardcoded light-hex values in base CSS)
- Reports amendment notice border
- Reports customer row border
- Picker back button (accent-tinted fill)
</what-built>
