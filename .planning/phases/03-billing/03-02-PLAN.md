Purpose: This is the first screen sellers see after signing in. The "New Sale" hero card must be unmissable and instantly tappable. Summary stat cards provide at-a-glance context. Live data wiring comes in Phase 5.

UX Design Principles Applied (UI/UX Pro Max):
- Primary action rule: One dominant CTA per screen — "New Sale" card is that CTA
- Bento grid: varied card sizes create visual rhythm without clutter
- Today card spans full width — most relevant data gets most space
- Icon per stat card — supports quick scanning without reading labels
- Tabular numeric figures for all revenue/count values
- Color system: blue (primary action), green (week/positive trend), purple (month/historical)
- Gradient accent on Today card — subtle depth, modern 2026 feel
- All text is static placeholder — no Firestore reads, no errors possible

Output: A clean 55+ line `modules/dashboard.js` that renders beautifully on first load and directs sellers to their primary task instantly.
