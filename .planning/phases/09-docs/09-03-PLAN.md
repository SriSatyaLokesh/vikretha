<task type="auto">
  <name>Task 1: Bundle size audit</name>
  <files></files>
  <action>
Measure the size of all app JavaScript files (excluding Firebase SDK which is loaded from CDN). Report the total and flag if it exceeds the 30 KB target.

Run the following PowerShell to measure:
```powershell
$files = Get-Item app.js, sw.js, lib/firebase-init.js, modules/*.js
$report = $files | Select-Object Name, @{N="Bytes";E={$_.Length}}, @{N="KB";E={[math]::Round($_.Length/1KB,2)}}
$total = ($files | Measure-Object Length -Sum).Sum
$report | Format-Table -AutoSize
Write-Host ""
Write-Host "Total app JS: $([math]::Round($total/1KB,2)) KB (target: < 30 KB)"
if ($total / 1KB -gt 30) { Write-Host "⚠️ OVER BUDGET — optimise before shipping" } else { Write-Host "✓ Within budget" }
```

Record the results in the SUMMARY.md. If total exceeds 30 KB, identify the largest file(s) and add a note in SUMMARY.md — but do NOT automatically minify/remove code (that requires user decision).

Note: `styles/main.css` is excluded from the JS budget — only .js files count. Firebase SDK (loaded from CDN by `lib/firebase-init.js`) does not count.
  </action>
