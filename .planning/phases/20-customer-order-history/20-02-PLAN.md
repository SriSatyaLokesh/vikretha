In `modules/receipt.js`, inside the `render()` function:

1. **Add conditional link to the template** — in the `container.innerHTML = \`...\`` block, inside `.receipt-actions`, add the customer history link between the WhatsApp button and the "← Back to Dashboard" anchor:
   ```html
   ${sale.customer_phone ? `
   <a id="btn-cust-history" href="#"
      style="display:block;text-align:center;margin-top:4px;color:var(--primary);font-size:0.875rem;font-weight:500;text-decoration:none;">
     👤 Customer history →
   </a>` : ''}
   ```

2. **Bind click handler** — after `container.innerHTML` is set (in the existing event binding section below the template), add:
   ```js
   if (sale.customer_phone) {
     const custHistBtn = document.getElementById('btn-cust-history');
     if (custHistBtn) {
       custHistBtn.addEventListener('click', e => {
         e.preventDefault();
         window.location.hash = '#/reports/customers/' + encodeURIComponent(sale.customer_phone);
       });
     }
   }
   ```
   Use `addEventListener` + `e.preventDefault()` rather than an inline `href` value to ensure proper encoding and prevent hash-injection (encodeURIComponent applied to `sale.customer_phone` before assignment — same pattern as Phase 15 receipt navigation).
