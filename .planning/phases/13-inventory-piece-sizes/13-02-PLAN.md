**1. Add `_showSizePicker(container, inv)` function** (new function, place after `_loadInventory`):

A bottom-sheet overlay (same sheet pattern as inventory.js modals) listing size variants:
```
Title: "Select Size — {inv.name}"
Body: list of size options rendered from Object.entries(inv.sizes)
```
Each option row shows:
- Size label (bold)
- Width if present: "W: {width}"
- PSI if present: "PSI: {psi}"
- Stock count: "{stock} in stock"
- Disabled (opacity 0.4, not tappable) if `sizeData.stock <= 0`

Tapping a size option:
1. Compute `cartKey = inv.id + '::' + sizeKey`
2. If `_cart.has(cartKey)` → increment qty: `_cart.get(cartKey).qty++`
3. Else → `_cart.set(cartKey, { id: inv.id, cartKey, name: inv.name, price: Number(inv.price), unit: 'pcs', qty: 1, sizeKey, sizeLabel: sizeData.label })`
4. Close the picker overlay
5. `_refresh(container)`

Include a "Cancel" button that closes without adding.

XSS: render size option text using `escapeHtml()` or `.textContent = ...`.
